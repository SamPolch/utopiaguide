# UTools

| |  | | --- | | **THIS PAGE** describes the UTools tool/addon for Utopia. If you have any queries with regards to the tool, direct your enquiries to the specific developer (refer below). | |

## What is uTools?

**Developer** Angelus

uTools is an Add-on for [Mozilla Firefox](Mozilla_Firefox.md) especially created for [Utopia](Utopia.md). It formats the Utopia pages and improves the general interface.

It may be downloaded at http://www.k3ltic.com/utools Compatible with new code

## Requirements

- A Mozilla Firefox Browser

## Features of uTools

| **Features List** | | |
| --- | --- | --- |
| - Ability to Redo spells and ops multiple times on the same province without having to re-select anything.  - Kingdom Pages may be sorted by all fields.  - Throne Page is formatted by TPA, WPA, OPA, DPA , pPA  - BPA on Science page  - Ability to Bookmark & Comment on Kingdoms for easy selection on the Mystics, Thievery, War Room & Kingdom Page.  - Ability to Mass Message.  - War Room Calculator.  - Ambush Calculator  - Inbuilt Forum Agent (with support of multiple agents)  - Able to generate SOT/SOM/SURVEY/SOS  - Foreman (stores and populates your build/strat)  - Cast multiple self-spells (removed)  - External kingdom paper formatter  - Various other features |
